0) Core assumptions

The speaker and microphone are separated (not co-located).

From IR alone, “mic→surface distance” is not uniquely solvable.

In Geometry Mode, with room + speaker + mic coordinates, the app can compute:

Which wall/floor/ceiling likely caused each peak

The reflection point on that surface

The mic→reflection point distance (a physically meaningful distance)

Optional 2nd order reflections (wall-wall, wall-floor, etc.) with limits

3) Image Source Method math (must implement)
3.1 First-order reflections (single plane)

For each plane (surface) with unit normal n and plane equation:

Plane defined by point P0 on plane and normal n (normalized).

Reflect the speaker point S across the plane to get image source S′:

Let:

v = S - P0

dist = dot(v, n)

S' = S - 2 * dist * n

Predicted reflection path length:

L_refl = |S' - M|
Direct path length:

L_dir = |S - M|
Predicted delay relative to direct:

Δt_pred = (L_refl - L_dir) / c (seconds)

Reflection point on the surface:

Ray from S′ to M: R(u) = S' + u*(M - S')

Intersect with plane: solve dot(R(u) - P0, n) = 0

u = dot(P0 - S', n) / dot(M - S', n)

If u not in [0,1] → intersection is outside segment; still compute but mark as “geometrically invalid”

P_ref = R(u)

Mic-to-surface distance for this reflection:

d_mic_refPoint = |M - P_ref|
(also optional speaker-to-refPoint: |S - P_ref|)

Also compute incidence angle (optional):

direction to plane from S: dir_in = normalize(P_ref - S)

incidence cos = abs(dot(dir_in, n))

3.2 Second-order reflections (optional)

Generate second-order image sources by reflecting S across plane A then plane B:

S1 = reflect(S, planeA)

S2 = reflect(S1, planeB)

Then predicted length |S2 - M|

You may compute two reflection points by back-tracing (advanced) OR just provide the two-surface label and predicted delay.
Keep it efficient:

Only compute second-order for a small set (e.g. top 4 most reflective surfaces by weight).

Or cap total predicted paths.

5) Matching algorithm: predicted reflections ↔ measured peaks

Implement a robust assignment:

Inputs

Measured peaks list Pk with (delay_ms, rel_dB, severity)

Predicted reflections list Rj with (surfaceLabel(s), Δt_pred_ms, reflectionPoint, validityFlag, priorWeight)

Cost function
For each pair (Pk, Rj):

timeErr = abs(Pk.delay_ms - Rj.Δt_pred_ms)

cost = timeErr / tol_ms

Add penalties:

if reflection point invalid → +0.75

if surface weight low → + (1 - weight)*0.5

If timeErr > tol_ms * 2 → disallow (infinite cost)

Assignment method

Prefer Hungarian algorithm for global optimal matching (implement or use a lightweight TS implementation).

If too heavy, use a greedy approach:

sort peaks by severity desc

for each peak choose the lowest-cost unused reflection candidate

mark as assigned

Provide a “confidence score”:

confidence = clamp(1 - timeErr/tol_ms, 0, 1)

downgrade if invalid geometry

Output classification

Assigned peaks: show surface + reflection point + mic distance

Unassigned peaks: show “Unknown / higher-order / object (desk/stand) / not in model”
