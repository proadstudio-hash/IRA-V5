Feature 3 — Clarity/Definition metrics: C50, C80, D50, Ts

Add a new report section “Clarity & Definition”.

Definitions (based on IR energy):

Let t=0 at direct arrival t0 (your alignment reference).

Let E_total = ∫ e(t) dt over a meaningful window (e.g., 0 to 500 ms or full length).

Let E_early50 = ∫_{0..50ms} e(t) dt, E_late50 = E_total - E_early50

Let E_early80 = ∫_{0..80ms} e(t) dt, E_late80 = E_total - E_early80

Compute:

C50 (dB) = 10 log10(E_early50 / E_late50)

D50 (%) = 100 * (E_early50 / E_total)

C80 (dB) = 10 log10(E_early80 / E_late80)

Ts (ms) = 1000 * (∫ t e(t) dt / ∫ e(t) dt) over the same window

Output values + short interpretation:

speech/vocal: C50, D50 helpful

music: C80

Ts indicates perceived “distance/blur”
